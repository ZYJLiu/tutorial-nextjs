# H1

<CustomCard linesToHighlight="1-6">

### Step 1

Add imports from @solana/web3.js

```js
import {
  Connection,
  Keypair,
  LAMPORTS_PER_SOL,
  clusterApiUrl,
} from "@solana/web3.js";
```

</CustomCard>

<CustomCard linesToHighlight="8-9">

### Step 2

Establish a connection to the Solana devnet cluster

```
const connection = new Connection(clusterApiUrl("devnet"), "confirmed");
```

</CustomCard>

<CustomCard linesToHighlight="11-12">

### Step 3

Generate a new keypair

```
const wallet_1 = new Keypair();
```

</CustomCard>

<CustomCard linesToHighlight="14-18">

### Step 4

Request the airdrop

```js
const transactionSignature = await connection.requestAirdrop(
  wallet_1.publicKey,
  5 * LAMPORTS_PER_SOL,
);
```

</CustomCard>

<CustomCard linesToHighlight="20-22, 24-32">

### Step 5

Get the most recent blockhash

```js
const { blockhash, lastValidBlockHeight } =
  await connection.getLatestBlockhash();
```

Confirm the airdrop transaction

```js
// Confirm the airdrop transaction
await connection.confirmTransaction(
  {
    blockhash,
    lastValidBlockHeight,
    signature: transactionSignature,
  },
  "confirmed",
);
```

</CustomCard>

<CustomCard linesToHighlight="34-36">

### Step 6

Retrieve and log the new balance of each wallet after the transfer

```js
const postBalance = await connection.getBalance(wallet_1.publicKey);
console.log("wallet_1 balance:", postBalance / LAMPORTS_PER_SOL);
```

</CustomCard>
