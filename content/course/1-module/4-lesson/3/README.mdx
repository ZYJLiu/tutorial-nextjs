# Create Token Account

## Step 2: Initialize Token Account

In this step, we will initialize the token account data by invoking the Token Program.

Since the instruction in the previous step transfers ownership of the account to the Token Program, the Token Program is able to modify the data on the account as specified in this instruction.

We can do this using the `createInitializeAccountInstruction` function from `@solana/spl-token`, which builds a `TransactionInstruction` that invokes the Token Program to initialize the token account.

```js
/**
 * Construct an InitializeAccount instruction
 *
 * @param account   New token account
 * @param mint      Mint account
 * @param owner     Owner of the new account
 * @param programId SPL Token program account
 *
 * @return Instruction to add to a transaction
 */
export function createInitializeAccountInstruction(
    account: PublicKey,
    mint: PublicKey,
    owner: PublicKey,
    programId = TOKEN_PROGRAM_ID
): TransactionInstruction {...}
```
